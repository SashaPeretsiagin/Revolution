Elasticsearch - движок полнотекстового поиска и аналитики.
может применяться для аналитической обработки больших массивов данных, анализа логов серверов и многих других задач.
т.е Elasticsearch — хороший выбор для сбора и анализа данных среднего объёма
--------
В нашем проект можно использовать для анализа логов серверов.
Так же можно использовать для поиска по geo координатам( например людей или групп),
поиск по особенностям группы игроков ( например уровень, вещи, и тд.),
поиск по статиски игрок (возможно анализ стиля игры),
поиск по количеству побед, поражений.

Для анализа групп в которые чаще вступаеют их особенности.

Можно собирать и анализировать статистику часто покупаемых предметов и делать
на основе этого модефикации.
Или же редко покупаемые придметы и деалть скидку на них.
