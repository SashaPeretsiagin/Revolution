UserService - сервис пользователей, отвечает за создание новых пользователей, их удаление, изменение, вход, выход
и просмотр других профилей.
Методы:
*createUser - создание нового пользователя
*loginUser - вход в систему
*logoutUser - выход из системы
*getUser - просмотр чужого профиля
*updateUser - обновить данные пользователя
*deleteUser - удалить пользователя

PayService - сервис покупки предметов из магазина
Методы:
*payForItem - оплата предмета из магазина с помощью кредитной
карты онлайн

ShopService - целевой сервис магазина, отвечающий за целевый действия на площадке по продажи игровых предметов, такие как
поиск предметов по названию, по типу, получение информации о предмете, добавление новых предметов в корзину, их удаление их удаление из корзины
Методы:
*addNewItemToTheBusket - добавить новый предмет в корзину
*findListOfItemsByType - получить список предметов по их типу
*findListOfItemsByName - получить список предметов с частичным или полным совпадением
названия предмета
*getItem - получить информацию о предмете
*deleteItemFromBusket - удалить игровой предмет из корзниы
*sortListOfItemsByParametr - сортирует список предметов по указанному пармаетру
